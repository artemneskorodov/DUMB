# **D**on't **U**se **M**any **B**rains (DUMB)

## Описание проекта

**Don't Use Many Brains (DUMB)** - это учебный компилятор для простого C-подобного языка
программирования высокого уровня.

## Архитектура

### Front-end

В фронтенде происходит лексический анализ, а также генерация абстрактного синтаксического дерева (AST)
с помощью алгоритма рекурсивного спуска. Далее происходит генерация **SSA IR**, который позволит произвести
достаточно много интересных оптимизаций, которые включают в себя сложные алгоритмы над графами.

### Optimizer

В этом модуле реализуются оптимизирующие алгоритмы над **SSA IR**:
- Constant Folding
- Dead Code Elimination
- Constant Propagation

Возможно также будут реализованы оптимизации **Loop Invariant Code Motion** (вынос инвариантов за цикл),
**Register Allocation** (распределение регистров) и **Sparse Conditional Constant Propagation** (разреженное
условное распространение констант, которое позволяет не только заменить использование переменных на
использование констант, но и анализировать условные ветвления).

### Backend

Этот модуль позволяет генерировать ассемблер (или исполняемый ELF - файл). В нем происходит управление
стеком вызовов.

## Описание языка

Для начала будет реализован простейший язык без типизации - все переменные будут иметь целочисленный тип.
Язык будет поддерживать локальные и глобальные переменные, функции, циклы, условия.
Далее, если это будет иметь учебную ценность, планируется реализовать типизацию (начиная с массивов) с
возможностью добавления пользовательских типов.
